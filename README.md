
<div align = "center">

This project is a responsive and dynamic website built for Source Flooring LTD, a flooring business started by Belal Aljoudy. The website serves as an online platform to showcase services, collect customer inquiries, and provide contact information.

</div>

<br>
<div style="width: 100%;">
    <a href="#"><img src="https://raw.githubusercontent.com/ferasaljoudi/AssetsRepository/main/BrownSVGs/overview.svg" alt="Overview" style="width: 100%"></a>
</div>
<br>

The Source Flooring LTD website provides:

- An easy-to-navigate interface for customers.
- A form to submit inquiries and request free quotes.
- A showcase of services offered by the business.
- Contact details with clickable phone and address links for convenience.

This website was developed as a passion project for my brother, Belal Aljoudy, to help him establish an online presence for his new business.
